概要：いろいろチェック付けたり選択したりしてHPの変化を楽しんでね
　　　攻撃ヒット時発生型（{{unitsList_onHit[0].unitInfo.unitName}}～{{unitsList_onHit[3].unitInfo.unitName}}）の攻撃間隔は海外版wikiの60fpsの値を元に計算したから誤差があるかもしれない
　　　自己バフはまだ実装してないから[追加]でクラスを修羅とか天魔天狗とかに設定しても再動短縮とかスキル時間延長とか反映されない 2021/05/27(木)追加 (漏れがあるかもしれない)
　　　グラフの色が見分けづらいのは仕様です すまん　(変更したいときは下の折り畳み参照)
　　　グラフは凡例の名前をクリックすると表示/非表示を切り替え可能
　　　グラフは右クリックから画像として保存できる模様
　　　不具合報告や意見・要望がある方はこちらから
　　　レイアウトのセンスがほしい

簡単な内容変更(コンソール)
F12キーで開発ツールを開き、コンソール(Console)タグの入力ウィンドウに以下を参考にコードを入力し実行
ページを更新するとすべてリセットされる
・初動短縮や再動短縮などのオプションのチェックボックスをアクティブにしたい。
　　vm.list[i].disable=false
　　　・list：初動短縮　rWTlist
　　　　　　 再動短縮　rCTlist
　　　　　　 スキル時間延長　skillExtendlist
　　　　　　 硬直短縮　編成バフ　hasteTeemList
　　　　　　 　　　　　スキル　　hasteSkillList
　　　・i　 ：リストの番号(一番上が0、順に1、2、･･･)
　　　　　　 「vm.list[i].buffer」で名前確認可能
　　(例)硬直短縮(編成バフ)の雷光の魔剣士タラニアのチェックボックスをアクティブにしたい。
　　　　→コンソールタグの入力ウィンドウに「vm.hasteTeemList[3].disabled=false」と入力してEnter

・グラフの非表示じゃ物足りない。ユニットデータごと消したい。
　　vm.unitsList_type.splice(i, 1)
　　　・type：スキル発動時発生型　onSkillAct
　　　　　　　 攻撃ヒット時発生型　onHit
　　　・i　　：リストの番号(一番上が0、順に1、2、･･･)
　　　　　　　 「vm.unitsList_type[i].unitInfo.unitName」で名前確認可能
　　(例)スキル発動時発生型の猫又コハルのデータを消したい。
　　　　→コンソールタグ(略)に「vm.unitsList_onSkillAct.splice(4, 1)」と入力して(略)

・グラフの色を変えたい。※実行した後、グラフにマウスオーバーすれば反映される。なにかしらオプションを変えるだけでリセットされるので注意。
　　vm.canvas.data["datasets"][i].borderColor='color'
　　　・i　　：リストの番号(一番上が0、順に1、2、･･･；発生型の区別なく通し番号なので注意)
　　　　　　　 「vm.canvas.data["datasets"][i].label」で名前確認可能
　　　・color：ggrks
　　　　　　　 カラーネーム；黒ならblack、赤ならredなどその色の名前
　　　　　　　 RGB；#+16進数6桁　R(赤)、G(緑)、B(青)の表現したい色に含む値をそれぞれ16進数2桁で表現(ゾロ目は3桁に短縮可能#112233→#123)　黒#000、赤#f00、緑#0f0
　　　　　　　 　　　rgb(r, g, b)　それぞれの値を10進数で0～255、または0～100%の範囲で指定　黒rgb(0, 0, 0)、赤rgb(255, 0, 0)、緑rgb(0%, 100%, 0%)
　　　　　　　 HSL；hsl(h, s, l)　H(色相)、S(彩度)、L(明度)。それぞれhは0～360、sとlは0～100%の範囲で指定　黒hsl(0, 0%, 0%)、赤hsl(0, 100%, 50%)、緑hsl(120, 100%, 50%)
　　　　　　　 透過；RGB16進数は+2桁(00～ff)、rgb(r, g, b)はrgba(r, g, b, a)、hsl(h, s, l)はhsla(h, s, l, a)とa(0～1)を追加　黒rgba(0, 0, 0, 1)、黒hsla(0, 0%, 0%, 0.7)
　　(例)グラフの犬神ツムジの色をいい感じに透過したツムジカラーにしたい。
　　　　→(略)「vm.canvas.data["datasets"][12].borderColor='rgba(0, 238, 255, 0.7)'」(略)
